由于环境变量 `ALIYUN_AK` 和 `ALIYUN_SK` 尚未加载到当前 Cloud Agent 会话，需要通过以下方式之一完成最后的部署步骤：

#### 选项 1: 手动部署（推荐 ⭐ - 最快速）

**所需时间：5-10 分钟**

1. 登录阿里云 OSS 控制台：https://oss.console.aliyun.com/
2. 创建 Bucket（名称：`photo-share-hub`，区域：杭州，权限：公共读）
3. 配置静态网站托管（首页：`index.html`，404页：`index.html`）
4. 上传 `/workspace/frontend/dist/` 目录下的所有文件
5. 访问：`http://bucket名.oss-cn-hangzhou.aliyuncs.com/index.html`

**就这么简单！** 🎉

#### 选项 2: 使用一键脚本

```bash
# 编辑脚本，填入 AccessKey
nano /workspace/one-click-deploy.sh

# 运行部署
./one-click-deploy.sh
```

#### 选项 3: 重启会话后自动部署

如果 Cursor Dashboard 的 Secrets 已配置，重启 Cloud Agent 后：

```bash
cd /workspace
./setup-deployment.sh  # 自动配置环境
cd frontend
./deploy.sh            # 自动部署
```
